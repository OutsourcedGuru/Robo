Add Printer Wizard
  • 1) Host and Hostname

    Below, you can change your printer's displayed name and IP address/hostname.

  • 2) Printer API Key

    You can use your phone's camera to scan the QR code of your printer's API key using the scan tool.

    The API key can be found on your printer's control screen under Utilities -> Network -> API QR Code. The QA code is also available on your printer's web dashboard under Settings -> API.

  • 3) Webcam URL

    You can optionally view your printer's webcam (if connected) via the app as well. This lets you monitor your print job. To do this, the app needs to know the URL of the webcam.

    If you know the webcam URL, you can enter it below or tap the button to have the app try to auto-detect the URL.

  • 4) Finalize Setup

    The final step in the setup process is allowing the app to detect specific settings of your printer's profile including whether it has a heated bed and how movement of the extruders are to be handled.

    Tap "Get Profile" to have this happen and once completed you will be returned to the main screen with your new printer ready to use.
