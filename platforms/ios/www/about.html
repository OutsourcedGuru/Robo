Robo app for iOS, Android and Microsoft Phone

[ ] with <3 by https://github.com/OutsourcedGuru

If you liked this app, please visit http://robo.fyi for great tips on using your printer.

  • Robo.fyi

    Robo.fyi is an informational website dedicated to the Robo 3D brand of printers especially for those who wish to source parts for modifications and upgrades.

  • github -> roboc2-scrabble-tiles

    Step-by-step instructions for printing a set of Scrabble tiles and holders.

  • github -> gettodahchoppa

    An executable to generate multiple sub-jobs from indicated GCODE file, suitable for printing color-by-layer effects.

  • github -> SlicingInfo

    A command line executable program to display slicing information from selected GCODE file.

  • github -> OctoPrint-slicing-info

    OctoPrint plugin to display slicing information from selected GCODE file.

  • github -> roboc2-cantilever-stability-upgrade

    Step-by-step instructions for upgrading the Robo C2 with cantilever stability for the print bed.

  • github -> roboc2-dual-spool-upgrade

    Step-by-step instructions for upgrading the Robo C2 for dual filament spools.

  • github -> OctoPrint-plugin-toggle-autolevel

    An OctoPrint plugin to allow G29 commands (auto-leveling) to be commented out.

  • github -> roboc2-dual-runout-upgrade

    Step-by-step instructions for upgrading the Robo C2 to support dual filament run-out detection.

  • github -> roboc2-samba-upgrade

    Step-by-step instructions for updating Samba configuration on your Robo C2 printer.

  • github -> roboc2-sound-upgrade

    Step-by-step instructions for adding sound events to your Robo C2 printer.

  • github -> roboc2-pi-noir-upgrade

    Step-by-step instructions for adding Pi NoIR webcam to Robo C2 printer.

  • github -> cura-filament-profiles

    A collection of custom material profiles ready to be imported into Cura.
