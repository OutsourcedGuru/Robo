Add Printer Wizard
info
  • 1) Host and Hostname

    Below, you can change your printer's displayed name and IP address/hostname.

  • 2) Printer API Key

    The API key can be found on your printer's control screen under Utilities -> Network -> API QR Code. The QA code is also available on your printer's web dashboard under Settings -> API.

  • 3) Webcam URL

    You can optionally view your printer's webcam (if connected) via the app as well. This lets you monitor your print job. To do this, the app needs to know the URL of the webcam.

    If you know the webcam URL, you can enter it below.

  • 4) Finalize Setup

    Click the Save button to complete your printer's profile.
