Add Printer Help

Host and Hostname

From this section, you can add your printer's name (serial number) and its hostname (so that this app may connect to it).

Your printer has a serial number on the back: serial-number as its name and serial-number.local as its hostname...

...which will work for anyone with an OS X or iOS computer or those who have installed iTunes (specifically the related Bonjour broadcast protocol from Appple).

For anyone else, for the hostname field they will need to use the dotted IP address of their printer which can be found in the LCD menu under Utilities -> Network.

Printer API Key

This section allows you to enter your printer's API key which is essential for permissioning you to gather information from it. You can find this under the LCD's Utilities -> Network -> API QR Code menu option.

Webcam URL

If you have one, this section will connect this app with your printer's optional webcam. You may then use this app to stream a live video feed from your printer.

Finalize Setup

This section will complete the profile for your new printer so that you can connect to it with this app.
